課程內容

能力，從實作中養成！！前端後端一手包

HTML5, CSS3

學習HTML與CSS，建構出良好的網頁的基礎，
在後續利用框架快速開發與接到後端時，
都可以很好的銜接、修改、維護。

B

Bootstrap

Bootstrap 是一個結合 HTML, CSS 和 JS的框架，
透過這個框架，可以快速打造出一個響應式的網頁，
也可以將預設CSS樣式修改成自己的風格。

  php 

PHP

使用 PHP 的 Laravel 框架開發系統後端功能，結合前端的 HTML，即可製作出一個簡易完整的網站系統。

關於 PROACTIVE 平台

因為在學習網頁設計的路上，遇到障礙也突破了許多難關，體會到如果彼此都能分享出自己的經驗，能有一個平台交流分享網頁設計相關技術，那將會幫助到許多想要進入這個領域的新手。 Proactive 這個服務也因此而生， 希望能成為聚集網頁設計相關資源的平台， 也邀請有興趣分享經驗者與我們一起，舉辦設計、前端技術相關課程分享。.
